### Ejemplo de conexión ANDROID contra docker
Android corriendo desde un emulador o desde un dispositivo por ADB hace referencia 127.0.0.1 a él mismo, por eso hay que poner la IP local de la máquina a la que llamamos

## Configuracion
Cambia las credenciales genéricas del archivo que se encuentra en res por la ip de tu máquina, el user y el pass de tu maria en docker

## Extra
La clase es utilitaria con lo que es exportable a otros proyectos
